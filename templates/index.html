ALS → ActiveHub Code Generation Utility

Prepares a PSG-ready CSV that instructs the code generation system how many unique ActiveHub access codes to create for each customer line item, when migrating international establishment (and D2C) customers from Active Learn to ActiveHub. Source files are read-only; three artefacts are produced: the request CSV, an exceptions CSV, and a run summary.
